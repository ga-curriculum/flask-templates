## About

In this module, students will learn how to use Flask templates to create dynamic web pages. They will explore key concepts such as rendering HTML templates, integrating Python logic into templates using Jinja, and handling data through Flask's request object.

## Content

| Lesson                                                                        | Skills                                                                                                 |

| [Setup](../setup/README.md)                                                    | Setting up the development environment.                                                                |
| [Flask: Web Applications vs APIs](../flask-web-applications-vs-apis/README.md) | Understanding the difference between web applications and APIs.                                        |
| [Rendering HTML Templates](../rendering-html-templates/README.md)              | Creating and rendering HTML templates using Flask and Jinja.                                           |
| [Programming Logic in Templates](../programming-logic-in-templates/README.md)  | Integrating Python logic, including loops and conditionals, into HTML templates using Jinja.           |
| [Sending and Receiving Data](../sending-and-receiving-data/README.md)          | Understanding form submissions, extracting data from mock requests, and simulating a CREATE operation. |
| [Wrap Up](../wrap-up/README.md)                                                | Reviewing key concepts and preparing for the next module.                                              |
